# Minishell

## Description

Minishell is a simple Unix shell implemented in C.
It reproduces basic Bash behavior including:
- interactive prompt
- signal handling
- command execution
- pipes and redirections (to be implemented)

## Instructions

```bash
make
./minishell
```

## Architecture

The project is divided into:
- lexer
- parser
- executor
- builtins
- signals
- libft
